# um-code

## URL shortening

Service that:		
1)  Accepts a URL and returns a shortened version
2)  Takes a shortened url and returns the original longer URL 							

## Run
- Python 3.10.6
- Install packages
```
pip install -r requirements.txt
```
- Run tests
```
python -m unittest test.py
```
